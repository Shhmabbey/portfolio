Built with the following technologies: Ruby, Ruby on Rails, React.js, Redux.js, HTML, CSS / SCSS, ActiveRecord, PostgreSQL, Jbuilder, JavaScript, Postman, Git, AWS, and Heroku.

Mango Music

Mango Music is a social media platform in which users can build connections over what they love the most, music! This program utilizes the MERN stack, as well as other technologies such as: Spotify API, Genius API and Lyrics Ovh API. Users can search for tracks, discover new tunes, listen to song previews, and most importantly, share their thoughts and start a discussion about a particular piece.
